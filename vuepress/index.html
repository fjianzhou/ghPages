注意事项

确保自定义组件的名称包含连字符或符合 PascalCase 命名规则。否则，它将被视为内联元素，并被包裹在一个

标签内，这将导致 hydration（Vue 的渲染子过程）不匹配，因为

不允许块元素放置在其中。

嵌套标题链接

themeConfig.sidebarDepth 自定义此行为。默认深度是 1，它提取 h2 标题。将其设置为 0 将禁用标题链接，最大值为2，同时提取 h2 和 h3 标题

也可以在页面中用 YAML 语法设置 sidebarDepth: 2

# 配置 Environment Variables

  • 在git中申请Personal access tokens 方便操作git项目代码
  • 在travis设置环境变量（Environment Variables）

为什么要配置环境变量

在travis中设置环境变量方便脚本调用 access_token git push -f https://${access_token}@github.com/xxxx/ghPages.git master:gh-pages

# 方法二

# 编写.travis.yml文件

language: node_js
sudo: required
node_js:
  - 8.11.3
branch: master
cache:
  directories:
    - node_modules
script:
  - npm run docs:build
after_success:
  - cd docs/.vuepress/dist
  - git init
  - git add -A
  - git commit -m 'deploy'
  - git push -f https://${access_token}@github.com/fjianzhou/ghPages.git master:gh-pages
